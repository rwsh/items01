# items01. Тестовое веб-приложение

Для создание проекта: 

**Устанавливаем:**

Node.js

MongoDB

**Для создания приложения:**

npm init

npm install express --save

npm install mongoose --save

npm install hbs --save

**Запуск приложения:**

node index.js
